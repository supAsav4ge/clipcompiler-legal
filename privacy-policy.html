🔒 Politique de Confidentialité - ClipCompiler

Dernière mise à jour : 8 juillet 2025
🛡️ Engagement de confidentialité : ClipCompiler fonctionne entièrement en local sur votre système. Nous ne collectons, ne stockons, ni ne transmettons vos données personnelles à des tiers.

1. Informations Générales

Cette politique de confidentialité explique comment ClipCompiler traite vos informations lorsque vous utilisez notre service de compilation et publication de vidéos.

2. Données Collectées

Type de Données Utilisation Stockage
Tokens TikTok OAuth Authentification avec TikTok API Local uniquement (fichier JSON)
Métadonnées de vidéos Upload et publication sur TikTok Transmises directement à TikTok
Logs d'utilisation Débogage et amélioration Local uniquement (fichiers .log)
Fichiers vidéo Traitement et upload Local, supprimés après upload

3. Données NON Collectées

  • ❌ Informations personnelles d'identification
  • ❌ Mots de passe TikTok
  • ❌ Données de navigation
  • ❌ Adresses IP
  • ❌ Données d'utilisation transmises à des serveurs externes

4. Utilisation des Données

Les données traitées par ClipCompiler sont utilisées exclusivement pour :

  • 🎬 Publier vos vidéos sur votre compte TikTok
  • 🔐 Maintenir votre session d'authentification
  • 🐛 Diagnostiquer les problèmes techniques (logs locaux)
  • 📊 Améliorer les performances de l'outil

5. Partage de Données

🚫 Aucun partage : ClipCompiler ne partage aucune donnée avec des tiers. Seules les données nécessaires sont transmises directement à TikTok via leur API officielle.

6. Stockage et Sécurité

  • Stockage local : Tous les fichiers restent sur votre système
  • Chiffrement : Communications HTTPS avec TikTok API
  • Tokens : Stockés dans un fichier JSON local sécurisé
  • Expiration : Les tokens expirent automatiquement

7. Données TikTok

Lorsque vous utilisez ClipCompiler :

  • Vos vidéos sont envoyées directement à TikTok
  • TikTok applique sa propre politique de confidentialité
  • ClipCompiler ne conserve aucune copie de vos vidéos
  • Les métadonnées (titre, description) sont transmises à TikTok

8. Droits de l'Utilisateur

Vous avez le droit de :

  • ✅ Supprimer les tokens stockés localement
  • ✅ Consulter les logs d'utilisation
  • ✅ Révoquer l'accès à TikTok à tout moment
  • ✅ Désinstaller complètement ClipCompiler

9. Rétention des Données

  • Tokens : Jusqu'à révocation ou expiration
  • Logs : Supprimés automatiquement après 30 jours
  • Vidéos : Jamais stockées durablement
  • Cache : Vidé à chaque redémarrage

10. Conformité Légale

ClipCompiler respecte :

  • 🇪🇺 RGPD (Règlement Général sur la Protection des Données)
  • 🇺🇸 CCPA (California Consumer Privacy Act)
  • 📋 Conditions d'utilisation de TikTok
  • 🔒 Bonnes pratiques de sécurité OAuth 2.0

11. Cookies et Tracking

🍪 Aucun cookie : ClipCompiler ne utilise aucun cookie, pixel de tracking ou technologie similaire.

  • Des évolutions légales
  • Des améliorations de sécurité
